# 综合态势总控页 V2 版布局重构计划

## 背景与目标

根据 `综合态势总控页_V2改版布局方案.md` 的设计规范，将现有的三列式布局彻底重构为**地图为核心背景，两侧悬浮 HUD 的战术界面**。

## 核心变更内容

### 1. 整体布局重构
- **[MODIFY] [CommandOverview.vue](file:///e:/%E6%B6%88%E9%98%B2%E6%90%9C%E6%8F%B4%E6%99%BA%E8%83%BD%E9%80%9A%E4%BF%A1%E6%8C%87%E6%8C%A5%E7%B3%BB%E7%BB%9FV3/web/src/views/CommandOverview.vue)**
  - 移除 `.main-area` 的 Flex 三列结构。
  - 将 `<TacticalMap />` 提取至底层，设为绝对定位，铺满全屏（`z-index: 1`）。
  - 新增上层交互容器（`z-index: 10`），采用 `pointer-events: none` 穿透点击，仅 HUD 面板恢复 `pointer-events: auto`。
  - 容器左右两侧通过 Flex 分布放置左侧 HUD 和右侧 HUD。

### 2. 左侧导航与状态顶栏
- **[NEW] [SideNav.vue](file:///e:/%E6%B6%88%E9%98%B2%E6%90%9C%E6%8F%B4%E6%99%BA%E8%83%BD%E9%80%9A%E4%BF%A1%E6%8C%87%E6%8C%A5%E7%B3%BB%E7%BB%9FV3/web/src/components/layout/SideNav.vue)**
  - 实现左侧独立竖向导航栏（宽度 72px），包含图标（态势、视频、通讯、设定等）。
- **[MODIFY] [TopCommandBar.vue](file:///e:/%E6%B6%88%E9%98%B2%E6%90%9C%E6%8F%B4%E6%99%BA%E8%83%BD%E9%80%9A%E4%BF%A1%E6%8C%87%E6%8C%A5%E7%B3%BB%E7%BB%9FV3/web/src/components/layout/TopCommandBar.vue)**
  - 移除原有的中央页面导航 Tabs。
  - 转变为只读的、极简的态势仪表盘（时间、任务、链路状态、已搜覆盖率）。

### 3. 左侧 HUD 面板（视频 + 传感器）
- **[MODIFY] [MainVideoWindow.vue](file:///e:/%E6%B6%88%E9%98%B2%E6%90%9C%E6%8F%B4%E6%99%BA%E8%83%BD%E9%80%9A%E4%BF%A1%E6%8C%87%E6%8C%A5%E7%B3%BB%E7%BB%9FV3/web/src/components/video/MainVideoWindow.vue)**
  - 调整宽高和样式，使其适应左侧 HUD 上方（360x270），移除无用的背景和边框，增加玻璃态（glassmorphism）效果。
- **[NEW] [SensorMatrix.vue](file:///e:/%E6%B6%88%E9%98%B2%E6%90%9C%E6%8F%B4%E6%99%BA%E8%83%BD%E9%80%9A%E4%BF%A1%E6%8C%87%E6%8C%A5%E7%B3%BB%E7%BB%9FV3/web/src/components/sensors/SensorMatrix.vue)**
  - 实现左侧 HUD 下方的环境传感器矩阵面板（温湿度、CO浓度、辐射等）。

### 4. 右侧 HUD 面板与底栏
- 右侧 HUD 直接复用现有的 `AlertList.vue`、`TaskList.vue` 和 `PersonnelCards.vue`，但在 `CommandOverview.vue` 中将其包裹在玻璃态容器中，脱离文档流浮动于地图之上。
- **[MODIFY] [BottomBar.vue](file:///e:/%E6%B6%88%E9%98%B2%E6%90%9C%E6%8F%B4%E6%99%BA%E8%83%BD%E9%80%9A%E4%BF%A1%E6%8C%87%E6%8C%A5%E7%B3%BB%E7%BB%9FV3/web/src/components/layout/BottomBar.vue)**
  - 宽度从 100% 改为自适应（避开左侧导航），增加全景浮动感。
